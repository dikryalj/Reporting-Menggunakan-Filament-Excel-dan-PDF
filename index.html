F · Apa itu Reporting

Data mentah, jadi sesuatu yang bisa dibaca orang

Reporting adalah proses mengubah baris-baris di database jadi bentuk yang siap dibaca: laporan Mahasiswa, laporan Dosen, rekap Mata Kuliah, atau jadwal Perkuliahan.

Format yang dipakai di modul ini
PDF
Excel
CSV
Excel enak dipakai kalau datanya mau diolah lagi — difilter, dihitung ulang, dibuat pivot. PDF enak untuk laporan yang tinggal dicetak atau diarsipkan apa adanya.

H · Export Excel

Kelas export: sumber datanya Mahasiswa

Generate dulu kelasnya lewat artisan, lalu isi method collection() supaya Laravel Excel tahu data mana yang mau diekspor.

terminal
$
app/Exports/MahasiswaExport.php
use App\Models\Mahasiswa;
use Maatwebsite\Excel\Concerns\FromCollection;

class MahasiswaExport implements FromCollection
{
    public function collection()
    {
        return Mahasiswa::all();
    }
}
Penjelasan Kode (Pahami Baris Per Baris)
  • use App\Models\Mahasiswa; — Memanggil model Mahasiswa agar class ini bisa mengakses data mahasiswa di database.
  • implements FromCollection — Menggunakan fitur bawaan Laravel Excel agar data diekspor langsung dari Collection (kumpulan data laravel).
  • collection() — Fungsi wajib dari FromCollection. Fungsi ini mengembalikan baris data apa saja yang akan dimasukkan ke file Excel.
  • Mahasiswa::all() — Mengambil seluruh data dari tabel mahasiswa di database untuk diekspor.
FromCollection cukup untuk data yang jumlahnya wajar. Kalau tabel Mahasiswa nanti berisi puluhan ribu baris, ganti ke FromQuery supaya query di-stream sedikit demi sedikit, bukan ditarik semua ke memori sekaligus.
I · Report Controller

Controller yang memicu unduhan

ReportController memanggil MahasiswaExport dan menyerahkannya ke facade Excel, lengkap dengan nama file hasil unduhan.

terminal
$
app/Http/Controllers/ReportController.php
use App\Exports\MahasiswaExport;
use Maatwebsite\Excel\Facades\Excel;

class ReportController extends Controller
{
    public function mahasiswaExcel()
    {
        return Excel::download(
            new MahasiswaExport,
            'laporan-mahasiswa.xlsx'
        );
    }
}
Penjelasan Kode (Pahami Baris Per Baris)
  • use App\Exports\MahasiswaExport; — Memanggil class export mahasiswa yang telah kita buat sebelumnya (Slide H).
  • use Maatwebsite\Excel\Facades\Excel; — Memanggil helper bawaan Laravel Excel untuk memproses download file.
  • mahasiswaExcel() — Fungsi controller yang akan dipicu oleh route web untuk memulai unduhan.
  • Excel::download(...) — Memerintahkan Laravel Excel untuk mengunduh class export tadi dengan nama file laporan-mahasiswa.xlsx.
Excel::download() otomatis menyetel header Content-Disposition pada response. Karena itu browser langsung memunculkan dialog simpan file, tanpa perlu diatur manual.
J · Route Excel

Satu baris yang menyambungkan URL ke controller

Tanpa route ini, method mahasiswaExcel() di controller tidak bisa diakses lewat browser sama sekali.

routes/web.php
Route::get('/report/mahasiswa/excel',
    [ReportController::class, 'mahasiswaExcel']
);
Penjelasan Kode (Pahami Baris Per Baris)
  • Route::get(...) — Menentukan route dengan metode GET yang berarti route ini diakses saat membuka link/URL di browser.
  • '/report/mahasiswa/excel' — Alamat URL yang akan dipanggil untuk mengunduh file Excel.
  • [ReportController::class, 'mahasiswaExcel'] — Mengarahkan request ke ReportController untuk memanggil method mahasiswaExcel().
Route ini sengaja dibuat GET, bukan POST. Method GET cocok untuk aksi yang cuma mengambil data, tanpa mengubah apa pun di database, dan bisa langsung dipanggil dari link atau dibuka lewat browser.
K · Template PDF

Blade sebagai kertas cetak

DomPDF merender file Blade biasa. Loop @foreach mengisi tabel dari data yang dikirim controller.

resources/views/report/mahasiswa.blade.php
<h2 align="center">Laporan Data Mahasiswa</h2>

<table border="1" width="100%" cellpadding="5">
  <tr>
    <th>No</th><th>NIM</th><th>Nama</th><th>Prodi</th>
  </tr>
  @foreach($mahasiswa as $m)
  <tr>
    <td>{{ $loop->iteration }}</td>
    <td>{{ $m->nim }}</td>
    <td>{{ $m->nama }}</td>
    <td>{{ $m->prodi }}</td>
  </tr>
  @endforeach
</table>
Penjelasan Kode (Pahami Baris Per Baris)
  • table border="1" ... — Menggunakan tabel HTML standar. DomPDF belum mendukung CSS Grid/Flexbox modern dengan sempurna, sehingga tabel HTML tradisional adalah pilihan paling aman.
  • @foreach($mahasiswa as $m) — Melakukan perulangan (looping) untuk setiap data mahasiswa yang dikirimkan oleh controller.
  • {{ $loop->iteration }} — Fitur bawaan Blade Laravel untuk menampilkan nomor baris secara berurutan mulai dari angka 1.
  • {{ $m->nim }} dan sejenisnya — Menampilkan nilai atribut NIM, Nama, dan Program Studi dari database.
Halaman ini dirender oleh DomPDF, bukan browser. CSS modern seperti flexbox atau grid sering tidak didukung, jadi tag <table> lawas seperti di atas justru pilihan paling aman.
L · Export PDF

Merender Blade jadi file .pdf

Method ini mengambil semua data Mahasiswa, mengirimnya ke template Blade, lalu langsung menawarkan file untuk diunduh.

app/Http/Controllers/ReportController.php
use Barryvdh\DomPDF\Facade\Pdf;

public function mahasiswaPdf()
{
    $data = Mahasiswa::all();

    $pdf = Pdf::loadView(
        'report.mahasiswa',
        ['mahasiswa' => $data]
    );

    return $pdf->download('laporan-mahasiswa.pdf');
}
Penjelasan Kode (Pahami Baris Per Baris)
  • use Barryvdh\DomPDF\Facade\Pdf; — Memanggil library DomPDF untuk merender template Blade HTML menjadi file PDF.
  • Pdf::loadView(...) — Fungsi untuk merender file blade report/mahasiswa.blade.php dan mengirimkan data mahasiswa ke dalamnya.
  • $pdf->download(...) — Memicu browser agar langsung mengunduh file PDF hasil render dengan nama laporan-mahasiswa.pdf.
Pdf::loadView() menerima nama file Blade dan array data, persis seperti view() biasa. Bedanya cuma hasil akhir yang di-stream sebagai PDF, bukan halaman HTML.
M · Route PDF

Route kedua, pola yang sama

Sama seperti route Excel — bedanya cuma method tujuan dan path URL-nya.

routes/web.php
Route::get('/report/mahasiswa/pdf',
    [ReportController::class, 'mahasiswaPdf']
);
Penjelasan Kode (Pahami Baris Per Baris)
  • '/report/mahasiswa/pdf' — Alamat URL yang akan dipanggil saat pengguna ingin mengunduh laporan berformat PDF.
  • mahasiswaPdf — Mengarahkan URL tersebut ke fungsi mahasiswaPdf() yang ada di dalam ReportController.
Dua route, dua tujuan berbeda, tapi satu pola. Kalau nanti nambah laporan Dosen atau Kelas, tinggal salin dua baris ini dan ganti path serta nama method-nya.
N · Menu Report di Filament

Dua tombol baru di atas tabel

Header action di Resource Mahasiswa memanggil kedua route di atas dan membukanya di tab baru — tidak keluar dari dashboard.

app/Filament/Resources/MahasiswaResource.php
use Filament\Actions\Action;

public static function getHeaderActions(): array
{
    return [
        Action::make('Export Excel')
            ->url('/report/mahasiswa/excel')
            ->openUrlInNewTab(),
        Action::make('Export PDF')
            ->url('/report/mahasiswa/pdf')
            ->openUrlInNewTab(),
    ];
}
Penjelasan Kode (Pahami Baris Per Baris)
  • use Filament\Actions\Action; — Memanggil kelas aksi Filament untuk membuat tombol baru.
  • getHeaderActions() — Method bawaan Filament Resource untuk menentukan tombol apa saja yang tampil di bagian atas (Header) tabel.
  • Action::make('Export Excel') — Membuat instansi tombol dengan tulisan "Export Excel".
  • ->url('/report/mahasiswa/excel') — Mengarahkan link tombol ke route Excel/PDF yang sudah kita buat sebelumnya.
  • ->openUrlInNewTab() — Membuka link download tersebut di tab browser baru agar halaman admin Filament tidak tertutup.
